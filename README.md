# Medical Inventory Management App

A lightweight project to track, manage, and audit medical supplies and inventory for clinics, hospitals, and pharmacies. This repository contains the application source, supporting documents, planning artifacts, and a demo video.

## Project Goals

- Maintain accurate stock levels and expiry tracking.
- Simplify procurement and re-order workflows.
- Provide audit logs for accountability.
- Support performance testing and scaling (see [Documents/Perfomance_Testing/README.md](Documents/Perfomance_Testing/README.md)).
